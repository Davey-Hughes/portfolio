# Gallery Directory

Add your photography images to this directory and they will automatically appear in your gallery!

## How It Works

The Leptos server automatically scans this directory for image files and displays them in the gallery page. No need to manually update code - just drop photos in here and refresh the page.

## Supported Formats

- JPG / JPEG
- PNG
- WEBP
- GIF

## File Naming

The filename (without extension) will be used as the image title with some smart formatting:
- `sunset-landscape.jpg` → "sunset landscape"
- `my_awesome_photo.jpg` → "my awesome photo"
- `Portrait_2024.jpg` → "Portrait 2024"

## Tips

- Photos will be displayed in alphabetical order by filename
- Mix portrait and landscape orientations for visual interest in the masonry layout
- Optimize images before uploading (aim for under 500KB each)
- Use descriptive filenames since they become the photo titles
- Example naming: `01-mountain-vista.jpg`, `02-city-night.jpg`, etc.

## Recommended Sizes

- Width: 1200-2400px
- Maintain original aspect ratios
- Use high quality JPEGs with 80-90% quality setting
