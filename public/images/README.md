# Images Directory

Place your photography images in this directory.

## Required Images

For the portfolio to display properly, add the following images:

1. **Homepage Category Hero Images**:
   - `portraits-hero.jpg` - Featured portrait photography showcase
   - `landscapes-hero.jpg` - Featured landscape photography showcase
   - `wildlife-hero.jpg` - Featured wildlife photography showcase
   - Recommended size: 1800x1200px or larger (landscape orientation)
   - Format: High-quality JPG

2. **Gallery Photos**: `photo1.jpg` through `photo9.jpg`
   - These are displayed in the masonry gallery grid
   - Recommended: Mix of portrait and landscape orientations for visual interest
   - Format: JPG or PNG
   - Note: Add more photos by updating the photos vector in src/app.rs

3. **Profile Photo**: `profile.jpg`
   - Your photographer profile picture for the About page
   - Recommended size: 800x1000px (portrait orientation)
   - Format: JPG or PNG

## Design Philosophy

The site uses a minimalist, photography-first approach:
- White/light backgrounds let your images stand out
- Clean typography and ample whitespace
- Subtle hover effects and transitions
- Images are the hero, not decorative elements

## Tips

- Use high-quality, optimized images (compress without losing quality)
- Mix of portrait/landscape orientations creates visual interest in gallery
- Category hero images should be your absolute best work
- Consider a slight grayscale or muted color treatment for consistency
- Use tools like ImageOptim, TinyPNG, or Squoosh to optimize file sizes
- Aim for images under 500KB each for optimal performance
